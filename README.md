# ccsk-kit

Claude Code Starter Kit — production-ready workflows, agents, and documentation for Claude Code.

This kit is installed via [@ccsk/cli](https://github.com/ccsk-org/ccsk-cli):

```bash
npm i -g @ccsk/cli
ccsk init
```

## Contents

- `CLAUDE.md` — primary Claude Code instructions
- `.claude/agents/` — sub-agent definitions
- `.claude/rules/` — workflow rules
- `.claude/commands/` — slash commands including `/ccsk-bootstrap`
- `docs/` — documentation skeleton
